# Source code links

use the `#source` tag e.g. like this:

```
#source foo.c
```

and it will be expanded into proper link.

There must be only one directive on each line.

# Markdown styles

  - see source code: :eyes:
  - watch-out: :heavy\_exclamation\_mark:
  - pro-tip: :bulb:
  - task: :wrench:
  - see example: :flashlight:
  - solution: :key:
